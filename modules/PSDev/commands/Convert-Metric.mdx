## DESCRIPTION

Function that converts speed metrics

## EXAMPLES

### EXAMPLE 1

```powershell
Convert-Metric -Gbph 40 -ShortUnits -RoundToNearestInteger
```

This commands converts "40 gigabyte per hour" to all the other metrics in the table and rounds the value.
