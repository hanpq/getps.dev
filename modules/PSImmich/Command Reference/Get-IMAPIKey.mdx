## SYNOPSIS

Retrieves Immich API keys

## SYNTAX

### list (Default)

```powershell
Get-IMAPIKey [-Session <ImmichSession>] [-ProgressAction <ActionPreference>] [<CommonParameters>]
```

### id

```powershell
Get-IMAPIKey [-Session <ImmichSession>] -Id <String[]> [-ProgressAction <ActionPreference>]
 [<CommonParameters>]
```

## DESCRIPTION

Retrieves one or more API keys from the Immich server.
Can retrieve all API keys or specific keys by ID.
API keys are used for programmatic access to the Immich API.

## EXAMPLES

### EXAMPLE 1

```powershell
Get-IMAPIKey
```

Retrieves all API keys for the current user.

### EXAMPLE 2

```powershell
Get-IMAPIKey -Id 'bde7ceba-f301-4e9e-87a2-163937a2a3db'
```

Retrieves a specific API key by its ID.

### EXAMPLE 3

```powershell
'key1-uuid','key2-uuid' | Get-IMAPIKey
```

Retrieves multiple API keys by piping their IDs.
