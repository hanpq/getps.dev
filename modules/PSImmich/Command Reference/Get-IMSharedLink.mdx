## SYNOPSIS

Retrieves shared links for asset sharing.

## SYNTAX

### list (Default)

```powershell
Get-IMSharedLink [-Session <ImmichSession>] [-AlbumId <Object>] [-ProgressAction <ActionPreference>]
 [<CommonParameters>]
```

### id

```powershell

 [<CommonParameters>]
```

## DESCRIPTION

Gets shared links that allow external access to assets and albums.

## EXAMPLES

### EXAMPLE 1

```powershell
Get-IMSharedLink
```

Gets all shared links.

### EXAMPLE 2

