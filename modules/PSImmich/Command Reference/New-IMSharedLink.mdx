## SYNOPSIS

Creates a new shared link for assets or albums.

## SYNTAX

### asset (Default)

```powershell
New-IMSharedLink [-Session <ImmichSession>] -AssetId <String[]> [-AllowDownload] [-AllowUpload]
 [-Description <String>] [-ExpiresAt <DateTime>] [-ShowMetadata] [-Password <SecureString>]
 [-ProgressAction <ActionPreference>] [<CommonParameters>]
```

### album

```powershell
New-IMSharedLink [-Session <ImmichSession>] -AlbumId <String> [-AllowDownload] [-AllowUpload]
 [-Description <String>] [-ExpiresAt <DateTime>] [-ShowMetadata] [-Password <SecureString>]
 [-ProgressAction <ActionPreference>] [<CommonParameters>]
```

## DESCRIPTION

Generates public sharing links with configurable permissions and expiration.
