## SYNOPSIS

Removes asset stacks.

## SYNTAX

```powershell
Remove-IMStack [[-Session] <ImmichSession>] [-Id] <String[]> [-Force] [-ProgressAction <ActionPreference>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION

Deletes stacks and ungroups the contained assets.
Assets remain in library.

## EXAMPLES

### EXAMPLE 1

```powershell
Remove-IMStack -Id 'stack-id'
```

Removes stack after confirmation.

Removes the specified stack

### EXAMPLE 2

```powershell
Remove-IMStack -Id @('stack-1', 'stack-2') -Force
```

Removes multiple stacks without confirmation
