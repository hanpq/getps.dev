## SYNOPSIS

Creates a new asset stack.

## SYNTAX

```powershell
New-IMStack [[-Session] <ImmichSession>] [-AssetIds] <String[]> [-ProgressAction <ActionPreference>]
 [<CommonParameters>]
```

## DESCRIPTION

Groups related assets into a stack with the first asset as primary.
Minimum 2 assets required.

## EXAMPLES

### EXAMPLE 1

```powershell
New-IMStack -AssetIds @('asset1', 'asset2', 'asset3')
```

Creates stack with asset1 as primary.
