## SYNOPSIS

Retrieves Immich workflows

## SYNTAX

### list (Default)

```powershell
Get-IMWorkflow [-Session <ImmichSession>] [-ProgressAction <ActionPreference>] [<CommonParameters>]
```

### id

```powershell
Get-IMWorkflow [-Session <ImmichSession>] -Id <String> [-ProgressAction <ActionPreference>]
 [<CommonParameters>]
```

## DESCRIPTION

Retrieves one or more workflows from the Immich server.
Can retrieve all workflows or a specific workflow by ID.
Note: Workflow functionality is in Alpha state as of Immich v2.3.1.

## EXAMPLES

### EXAMPLE 1

```powershell
Get-IMWorkflow
```

Retrieves all workflows available to the current user.

### EXAMPLE 2

```powershell
Get-IMWorkflow -Id 'bde7ceba-f301-4e9e-87a2-163937a2a3db'
```

Retrieves a specific workflow by its ID.

### EXAMPLE 3

```powershell
'workflow1-uuid', 'workflow2-uuid' | Get-IMWorkflow
```

Retrieves multiple workflows by piping their IDs.
