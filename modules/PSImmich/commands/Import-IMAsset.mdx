## SYNOPSIS

## SYNTAX

```powershell
Import-IMAsset [[-Session] <ImmichSession>] [-FilePath] <FileInfo[]> [[-Duration] <String>] [-isArchived]
 [-isFavorite] [-isOffline] [-isReadOnly] [-isVisible] [[-libraryId] <String>]
 [-ProgressAction <ActionPreference>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION

Imports an Immich asset

## EXAMPLES

### EXAMPLE 1

```powershell
Import-IMAsset -FilePath C:\file.jpg
```

Uploads image to Immich
