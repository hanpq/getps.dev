## SYNOPSIS

Removes a shared link.

## SYNTAX

```powershell
Remove-IMSharedLink [-Session <ImmichSession>] -id <String[]> [-ProgressAction <ActionPreference>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## DESCRIPTION

Deletes shared link and revokes external access to associated assets.

## EXAMPLES

### EXAMPLE 1

```powershell
Remove-IMSharedLink -Id 'link-id'
```
