## SYNOPSIS

Removes assets from a stack.

## SYNTAX

```powershell
Remove-IMStackAsset [[-Session] <ImmichSession>] [-StackId] <String> [-AssetId] <String> [-Force]
 [-ProgressAction <ActionPreference>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION

Removes specific assets from an existing stack.
If primary asset is removed, another becomes primary.

## EXAMPLES

### EXAMPLE 1

```powershell
Remove-IMStackAsset -StackId 'stack-id' -AssetId 'asset-id'
```

Removes asset from stack after confirmation.

### EXAMPLE 2

```powershell
-AssetId <assetId>
```

Removes the specified asset from the stack

### EXAMPLE 3

```powershell
-AssetId <assetId> -Force
```

Removes the asset from the stack without confirmation
