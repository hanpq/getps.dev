## DESCRIPTION

Search for assets using smart search.
Do note that this uses the machine learning model find assets based on content so results may vary.
If you rather want to search based on metadata use the Find-IMAsset cmdlet.

## EXAMPLES

### EXAMPLE 1

```powershell
Search-IMAsset -Query 'Road'
```

Searches for assets matching content Road.
