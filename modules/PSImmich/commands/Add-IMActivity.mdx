## SYNOPSIS

## SYNTAX

```powershell
Add-IMActivity [[-Session] <ImmichSession>] [-albumId] <String> [[-assetId] <String>] [[-comment] <String>]
 [[-type] <String>] [-ProgressAction <ActionPreference>] [<CommonParameters>]
```

## DESCRIPTION

Adds a new activity to an album

## EXAMPLES

### EXAMPLE 1

```powershell
-AssetId <assetid> -Comment 'Great picture!' -Type comment
```

Adds a new comment to an asset in the specified album
