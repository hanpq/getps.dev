## SYNOPSIS

Creates a new Immich user account.

## SYNTAX

```powershell
New-IMUser [[-Session] <ImmichSession>] [-Email] <String> [[-Notify] <Boolean>] [-Name] <String>
 [-Password] <SecureString> [[-QuotaSizeInBytes] <Int64>] [[-ShouldChangePassword] <Boolean>]
 [[-StorageLabel] <String>] [-ProgressAction <ActionPreference>] [<CommonParameters>]
```

## DESCRIPTION

Adds new user with email, name, and notification preferences.

## EXAMPLES

### EXAMPLE 1

```powershell
$Password = Read-Host -Prompt 'Password' -AsSecureString
New-IMUser -Email 'testuser@domain.com' -Name 'Test User' -Password $Password
```

Creates new Immich user
