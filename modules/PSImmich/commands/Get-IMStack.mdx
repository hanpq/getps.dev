## SYNOPSIS

Retrieves asset stacks.

## SYNTAX

### list (Default)

```powershell
Get-IMStack [-Session <ImmichSession>] [-PrimaryAssetId <String>] [-ProgressAction <ActionPreference>]
 [<CommonParameters>]
```

### id

```powershell

```

## DESCRIPTION

Gets stacks that group related assets together (like photo bursts or HDR sets).

## EXAMPLES

### EXAMPLE 1
